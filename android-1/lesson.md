#### Вступ

Мова Dart лежить в основі фреймворку Flutter. Сучасний фреймворк, такий як Flutter, потребує високорівневої сучасної мови, щоб бути здатним забезпечити найкращий досвід розробнику та зробити можливим створення чудових мобільних додатків. Розуміння Dart є фундаментальним для роботи з Flutter; розробникам необхідно знати походження мови Dart, як спільнота працює над нею, її сильні сторони та чому вона є обраною мовою програмування для розробки з Flutter.

Ми розглянемо наступне:
- Знайомство з принципами та інструментами мови Dart 
- Розуміння того, чому Flutter використовує Dart
- Вивчення основ структури мови Dart
- Знайомство з ООП у Dart

Мова Dart, розроблена Google, — це мова програмування, яку можна використовувати для розробки веб-, десктопних, серверних та мобільних додатків. Dart — це мова програмування, що використовується для написання коду додатків Flutter, що дозволяє забезпечити найкращий досвід розробнику для створення високорівневих мобільних додатків. Отже, давайте дослідимо, що надає Dart і як вона працює, щоб ми могли пізніше застосувати отримані знання у Flutter.

Dart поєднує в собі переваги більшості сучасних високорівневих мов, пропонуючи такі зрілі функції:
- Продуктивний інструментарій: засоби для аналізу коду, плагіни для IDE та величезна екосистема пакетів.
- Збирання сміття (Garbage collection): автоматичне керування пам'яттю та видалення об'єктів, що більше не використовуються.
- Анотації типів (опціонально): для тих, хто прагне безпеки та чіткого контролю даних у додатку.
- Статична типізація: попри опціональність анотацій, Dart є типобезпечною мовою і використовує виведення типів (type inference) під час виконання. Це критично важливо для виявлення багів ще на етапі компіляції.
- Портативність: код можна не лише транспілювати в JavaScript для вебу, а й нативно компілювати під архітектури ARM та x86.

З моменту свого анонсу у 2011 році Dart невпинно розвивається. Важливим етапом став вихід версії Dart 2.0 наприкінці 2018 року:
- Зміна пріоритетів: спочатку мова створювалася для вебу як заміна JavaScript, проте зараз фокус змістився на мобільну розробку та Flutter.
- Вирішення проблем JavaScript: на відміну від JS, Dart пропонує надійність, властиву стабільним мовам програмування , виступаючи як його вдосконалений наступник.
- Висока продуктивність: інструменти Dart оптимізовані для великомасштабних проєктів, забезпечуючи швидкість роботи при збереженні гнучкості динамічної мови.
- Гнучкість та надійність: завдяки поєднанню ООП та опціональної типізації, Dart зберігає ідеальний баланс між цими двома характеристиками.
